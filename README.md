# Retail Sales EDA (OIBSIP Internship)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover trends, customer behavior, and business insights.

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Monthly sales trend analysis
- Product category performance
- Customer segmentation (Age & Gender)
- Correlation analysis

## 📈 Key Insights
- Electronics and Clothing generate the highest revenue
- Customers aged 26–35 spend the most
- Female customers contribute slightly more sales
- Higher quantity purchases increase revenue

## 💡 Business Recommendations
- Focus marketing on top-performing categories
- Target high-spending age groups
- Offer bulk purchase discounts
- Plan campaigns based on seasonal trends

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
Retail Sales Dataset (Kaggle)
